What it does

Built for cinephiles
who share.

01

Media Discovery

Search and browse thousands of movies and TV series powered by the TMDb API. Trending, popular, and curated collections at your fingertips.

02

Personal Lists

Maintain a Favorites collection and a Watchlist. Track what you've seen, queue what's next, and never lose a recommendation.

03

Social Recommendations

Send direct recommendations to friends with a personal note. Rate received picks on a five-star scale and spark threaded discussions.

04

Real-time Updates

Comments, likes, and notifications powered by Supabase Realtime. Live interaction without manual refreshes.

05

Secure Backend

PostgreSQL with Row Level Security (RLS) ensures every user only sees what they're authorised to access. Auth with custom profile triggers.

06

Cinematic UI

High-contrast dark theme with NativeWind (Tailwind CSS for React Native) styling. Optimised for readability and engagement on any device.

Download

Try it on your
Android device.

No need to clone the repo. Download the APK directly from the Expo build dashboard and install it on any Android device. No Play Store required.

Android required Enable "Install from unknown sources" in your device settings before installing.
No iOS build yet iOS users can run the app via Expo Go by cloning the repo and starting the dev server.
Download APK

How to install

01

Tap the Download APK button and open the Expo build page.

02

Click Download build artifact to get the .apk file.

03

Transfer the file to your Android phone and open it to install.

04

If prompted, allow installation from unknown sources and you're good to go.

Get started

Up and running
in minutes.

01

Clone & install

Clone the repository and install dependencies with npm.

git clone https://github.com/Pabl0Parra/RawCut cd RawCut npm install
02

Configure environment

Copy the example env file and fill in your Supabase and TMDb credentials.

cp .env.example .env
03

Set up the database

Run supabase/schema.sql in your Supabase SQL editor to scaffold tables, RLS policies, and triggers.

04

Start the dev server

npx expo start

Scan the QR with Expo Go, or press a / i / w for emulators.

Environment variables

EXPO_PUBLIC_SUPABASE_URL Your Supabase project URL (Settings → API)
EXPO_PUBLIC_SUPABASE_ANON_KEY Supabase anonymous public key
EXPO_PUBLIC_TMDB_API_KEY TMDb API v3 key — free at themoviedb.org
EXPO_PUBLIC_TMDB_IMAGE_BASE https://image.tmdb.org/t/p/w500
